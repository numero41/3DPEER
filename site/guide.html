3DPEER

Guide

What it does

3DPEER turns a 3D model into a single .html file. The model, its decompressor and its viewer travel together: the file opens with a double-click, works offline, and will still work years from now. Nothing is ever uploaded — every step runs in your browser.

Supported formats

Drop a glb, gltf, obj, stl, ply, fbx or usdz file. Non-glTF formats are brought in automatically. Fidelity on materials and textures can vary with how the original file was produced.

Compression

Everything is compressed locally, on your machine. The defaults are visually safe; the estimated output size updates as you move the sliders in export settings:

  • geometry precision — how finely the shape is kept. Lower is smaller; very low can look faceted.
  • shading precision — how finely surface shading is kept.
  • texture size / quality — cap texture size and quality. Quality 100 keeps your originals.
  • decimate — removes a share of triangles. Off by default: your geometry is never touched unless you ask.
  • reset — restores the default quality.

Sharing the file

The exported file is self-contained — send it like any attachment (mail, WhatsApp, Drive, Slack). Two mail-client behaviours to know about, both by design of the mail providers:

  • Gmail on desktop shows HTML attachments as source text in its preview. Recipients should download the file, then open it — it launches in their browser.
  • iPhone Mail previews attachments with scripts disabled: the recipient sees the built-in snapshot views and a short notice instead of the interactive model. Opening the file in a real browser restores full interaction.

Each export embeds a four-view snapshot grid, so even script-blocked previews show your model.

Viewer controls in the file

When "ship viewer controls" is checked, the exported file carries the on-screen controls — camera views, material and lighting options, and any notes you added. Uncheck it for a minimal orbit-only file.

Privacy

The site has no upload endpoint. Models never leave your device; the exported file makes zero network requests, ever.
